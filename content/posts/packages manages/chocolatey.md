## 命令

```sh
# 搜索软件包
choco search 软件名

# 安装软件包
choco install 软件包名

# 更新已安装的软件包
choco upgrade 软件包名

# 卸载软件包
choco uninstall 软件包名

# 查看已安装的软件包
choco list

```
