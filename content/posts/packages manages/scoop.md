## 命令

```sh
# 搜索
scoop search <app_name>

# 卸载
scoop uninstall <app_name>

# 查看已安装
scoop list

# 更新 scoop 和 bucket 
scoop update

# 更新所有
scoop update *

# 禁止更新指定app
scoop hold <app_name>

# 解除禁止更新app
scoop unhold <app_name>

# 清理旧版本 
scoop cleanup <app_name>

```
