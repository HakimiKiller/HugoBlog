## usage

```sh
# 安装特定版本
fvm install <version> --setup

# 查看已安装的 Flutter 版本
fvm list

# 全局切换 Flutter 版本
fvm global <version>

# 卸载 Flutter 版本
fvm remove <version>

# 列出可用的 Flutter 版本
fvm releases

# 配置
fvm config
```
