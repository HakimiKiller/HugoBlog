## usage

```sh
# 查看本地已安装的所有 Node.js 版本
nvm list

# 查看网络上可以安装的所有 Node.js 版本
nvm list available

# 安装指定的 Node.js 版本 (例如 nvm install 16.20.0)
nvm install <version>

# 切换到指定的 Node.js 版本
nvm use <version>

# 显示当前正在使用的 Node.js 版本
nvm current

# 卸载指定的 Node.js 版本
nvm uninstall <version>

```
